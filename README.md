## 💻 Local Portfolio Setup & Development

This repository contains the source code for my updated portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18+) installed.

### Setup Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/omtrivedioo3/DiyaPortfolio.git
   cd DiyaPortfolio
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run in Development Mode**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173/](http://localhost:5173/) to preview your changes locally.

4. **Build for Production**:
   ```bash
   npm run build
   ```
   This compiles the project into the `dist/` directory, optimized and ready for deployment.
